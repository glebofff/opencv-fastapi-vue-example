# opencv-fastapi-vue-example
# backend
## installation
```shell
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip3 install -r requirements.txt
```

## dev server
```shell
cd backend
uvicorn main:app --reload
```

# frontend
## installation
```shell
cd frontend
npm i
```

## dev server
```shell
cd frontend
npm run serve
```
